# Momontom

It's clone of the chrome app Momentum on Vanilla JS.<br>
Made with help of teacher Nicolás Serrano Arévalo in Nomad Coders.

# Features

<ul>
 <li>Real-Time Clock</li>
 <li>To-Do-List</li>
 <li>Weather of Current Location</li>
 <li>Random background</li>
</ul>
